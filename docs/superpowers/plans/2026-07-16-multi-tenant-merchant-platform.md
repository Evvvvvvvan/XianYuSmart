# XianYuSmart 多租户商家运营平台实施计划

1. 新增 Flyway 迁移：租户字段、商品自动化字段、运营资源、任务和分销表。
2. 配置 MyBatis 租户拦截器，补齐账号租户归属和后台写入归属。
3. 为商品 DTO、Mapper、Service 和 Controller 增加自动评价/擦亮配置接口。
4. 扩展平台 API 调用工具的接口版本能力，实现评价与擦亮执行服务及调度器。
5. 实现运营资源、任务、分销的租户级 CRUD、立即执行和定时调度。
6. 实现素材转发布、货源转素材、规则生成任务、结果回写和补偿重试。
7. 在商品管理页增加两个商品级自动化开关。
8. 新增运营中心页面和导航，覆盖地址、素材、发布、采集分销、返佣、任务、通知与风控。
9. 使用临时测试验证租户过滤、开关过滤、任务状态和响应解析，验证完成后删除临时测试。
10. 执行 Maven 测试、前端类型检查与构建、敏感信息扫描、差异自审。
11. 提交并推送 `main`，随后调用项目本地部署流程更新生产服务并验证健康状态。
